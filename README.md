SplitSprites
============

SplitSprite is a tool which can split the plist file(Supported by TexturePacker and cocos2d) into multiple textures 


Usageï¼š

SplitSprites -i=[inputPlistFile|inputDir] [-o=.|outputDir] [-c=true|false] [-e=true|false]

-i input plist file or this directory which contains multiple plist file

-o output directory

-c compact mode

-e export or not xml file which indicates the info of all the sprites
